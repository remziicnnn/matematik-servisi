# Matematik Servisi API

Matematik Servisi, Django Rest Framework (DRF) kullanarak oluşturulmuş bir API'dir. Bu API, temel matematiksel işlemleri gerçekleştirmek için çeşitli endpoint'ler sunmaktadır.

## Özellikler

- Toplama
- Çıkarma
- Çarpma
- Bölme
- Faktöriyel hesaplama
- Asal kontrol
- EBOB (En Büyük Ortak Bölgen) hesaplama
- EKOK (En Küçük Ortak Kat) hesaplama
- Matris çarpma
